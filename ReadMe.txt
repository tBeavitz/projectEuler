Project Euler Read Me

Language: Python 3
Dependancies: Numpy 

Launch Options:
'test' - will use parameters that execute faster than the ones official parameters
0 -  Will execute all solved problems
(any positive integer) - will execute just that problem if a solution exists

File Descriptions:
projectEuler.py - The main hub of my Project Euler system
eulerFunctions.py - The location of generalized functions used to solve problems
eulerProblemInfo.py - Object that holds and builds the problem text and parameters
eulerArena.py - The testing ground
problems.txt - the raw problem text, also used to determine how many problems have been solved
